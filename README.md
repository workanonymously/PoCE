# POCE
1. Here we have combined all the frameworks to run in a single docker image in Combined_frameworks folder
2. If needs to run on subset, each folder has separate test.json file and in the dataset/main.json has all the files 
3. For initial poc generation choose to run python3 main.py --frameworks <framework_name>
4. For variant, choose to run python3 main.py --variant <framework_name>
